# 05 — Экраны и маршруты

## Основные экраны

### Главный экран / История

- список последних событий в карточках
- быстрый поиск («где ел пасту»)
- блок «важное сейчас»: ближайшие напоминания/долги/сроки
- быстрые фильтры по категориям
- плавающая кнопка создания (FAB)

### Добавление (New Entry Wizard)

- пошаговый ввод
- AI‑черновик
- редактирование и сохранение
- быстрый доступ к типам ввода (текст, фото, ссылка, голос)

### Поиск и AI‑чат

- строка поиска
- диалоговый режим
- выдача карточек и фильтров

### Карточка события

- ключевые поля вверху
- дополнительный контекст свернут
- вложения внизу
- действия: редактировать, напоминание, поделиться

### Список (List)

- позиции чека/набор записей
- итог по списку
- быстрые действия: добавить/распознать

### Lifeline (линия жизни)

- ключевые события по времени
- быстрое возвращение к важным периодам

### Настройки

- базовые настройки и приватность
- экспорт/импорт
- ссылки на предпочтения и напоминания

## Предложенные маршруты (PWA)

> Маршруты описаны логически; реализация зависит от фронта (Svelte 5 / SvelteKit).

- `/` — история (лента)
- `/new` — визард создания
- `/enter` — быстрый ввод (минимальный экран)
- `/entries` — список/фильтры (расширенный список)
- `/entry/:id` — карточка события
- `/list/:id` — список записей (например, чек)
- `/search` — поиск
- `/bot` — AI‑чат
- `/lifeline` — линия жизни
- `/settings` — настройки
- `/preferences` — предпочтения (категории, рецепты, поля)
- `/reminders` — напоминания
- `/planner` — планировщик (календарь напоминаний и намерений)
- `/groups` — совместные пространства
- `/dashboard` — сводка/аналитика (после MVP)

## Навигация (mobile‑first)

- быстрый доступ к созданию (FAB) в «зоне большого пальца»
- жесты (свайп) дублируются кнопками

## UX‑конвенции

- крупные кликабельные элементы (минимум 44×44 px)
- короткие тексты, один фокус на экран
- прогрессивное раскрытие деталей
- единообразие кнопок и иконок
