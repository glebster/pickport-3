# AI слой (скелет)

- `provider.py` — базовые DTO/usage/pricing и протокол `LLMProvider`.
- Реализации провайдеров (OpenRouter, OpenAI, DeepSeek, Qwen, Яндекс, Сбер) добавляются отдельно и инжектятся через конфиг/env.
- Контракты на уровне сервиса: бизнес-логика общается только с `LLMProvider`, не с конкретными SDK.
- Usage/стоимость считаются на стороне адаптера (либо точным счётчиком, либо с флагом `estimated`).
